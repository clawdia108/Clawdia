## Jak napsat ranní briefing

```
Dobré ráno ☀️ [den česky] [datum]

[1-3 věty: co je dneska hlavní a proč. Normální česká řeč.]

[Schůzky/cally s časem — jen důležité. Pokud jich je moc, řekni
"přesuň X na Y".]

[Čísla — jen když se něco změnilo. Zaokrouhleně. S kontextem proč
to říkáš.]

[Heads up — jen pokud něco hoří. Řekni co s tím.]
```

## Jak napsat večerní recap

```
Večerní shrnutí 🌙 [den česky] [datum]

[Co se dneska povedlo — konkrétně, s výsledky. 2-4 řádky.]

[Co se nepodařilo / přesouvá na zítra — řekni proč. 1-2 řádky.]

[Co potřebuješ rozhodnout — max 2-3 body.]
```

## Příklad dobrého ranního briefingu

```
Dobré ráno ☀️ úterý 4. 3.

Dneska máš 3 follow-upy — SCENOGRAFIE, Mycroft a DPP. Mycroft je
nejdůležitější, čekají na tvůj návrh od pátku.

Odpoledne: demo Friendly Stores ve 13:30. Ráno: VOP Group v 10:00.
Víc toho nestíhej, zbytek přesuň na pondělí.

Pipeline drží na 1,1M. Ale FNUSA a ProCare čekají přes týden —
ozvi se jim dneska.

Automatizace běží ručně, večer je restartuj (5 min).
```

## Příklad dobrého večerního recapu

```
Večerní shrnutí 🌙 úterý 4. 3.

Mycroft dostal návrh — CEO ho otevřel do hodiny, dobrý signál.
SCENOGRAFIE potvrdili kickoff na čtvrtek. DPP bez odpovědi zatím.

Demo s Friendly Stores se přesunulo na pátek (CEO měl kolizi).
VOP Group proběhlo dobře, posílají interní schválení.

Rozhodnutí: chceš na čtvrtek SCENOGRAFIE jet sám, nebo s Petrem?
```
