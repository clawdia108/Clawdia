# Daily Ops Playbook (v1)

## Morning block (30–45 min)
1. Review overdue + due-today activities
2. Build Top 3 priorities (A/B/C)
3. Flag stale deals (no movement > X days)
4. Confirm every active deal has next activity

## Midday block (15 min)
1. Check progress against Top 3
2. Resolve blockers
3. Re-prioritize remaining day

## Afternoon block (20–30 min)
1. Close loop on completed activities
2. Move deal stages where justified
3. Write PM summary and tomorrow’s first 3 actions

## Hygiene rules
- No active deal without next activity
- No ambiguous owner on deals/leads
- Mandatory source/qualification fields on new leads
- Duplicate checks daily
