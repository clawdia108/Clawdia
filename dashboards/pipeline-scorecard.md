# Pipeline Scorecard (v1)

## Core KPIs
- Active deals count
- Deals with next activity (% )
- Overdue activities count
- Stale deals count (>X days)
- Stage conversion rates
- Weighted pipeline value

## Lead KPIs
- New leads per week
- Qualified lead ratio
- Lead response time
- Lead-to-deal conversion

## Deal Health Signals
- Activity freshness
- Stakeholder coverage
- Stage momentum
- Risk flags
